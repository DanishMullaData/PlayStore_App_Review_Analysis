Play Store App Review Analysis
Overview:
This project analyzes Google Play Store apps and user reviews to extract insights on app ratings, categories, and installs using data analysis and visualization techniques.

Objectives:
Perform Exploratory Data Analysis (EDA)
Clean and preprocess app data
Visualize relationships between ratings, installs, and categories
Identify trends and patterns across app types

Tools and Libraries Used:
Python
Pandas
NumPy
Matplotlib
Seaborn
Plotly

Notebook:
Play_Store_App_Review_Analysis_Danish_Module3.ipynb

Dataset:
If you used a dataset, for example PlayStore.csv, it can be added inside a folder named data.
